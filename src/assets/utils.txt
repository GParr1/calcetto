| ------------- | ------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------- | --------------------------------- | ------------------------------------- |
| `button`      | Componente cliccabile che esegue un'azione              | Usato per bottoni, come un `<Button />` o `TouchableOpacity`.                                               | `accessibilityRole="button"`      | `<button>`                            |

| `heading`     | Intestazione di un contenuto, come `<h1>`, `<h2>`, etc. | Per titoli o sezioni principali del contenuto (titoli di articoli, testi, ecc.)                             | `accessibilityRole="heading"`     | `<h1>`, `<h2>`, ecc.                  |
| `text`        | Testo generico                                          | Per contenuti di testo generico.                                                                            | Nessuno specifico                 | `<p>`, `<span>`, `<div>`              |
| `img`         | Immagine                                                | Usato per immagini.                                                                                         | `accessibilityRole="img"`         | `<img>`                               |
| `navigation`  | Contenitore di navigazione                              | Usato per il menu di navigazione o per altre barre di navigazione.                                          | `accessibilityRole="navigation"`  | `<nav>`                               |
| `list`        | Contenitore di lista                                    | Usato per liste di oggetti.                                                                                 | `accessibilityRole="list"`        | `<ul>`, `<ol>`                        |
| `listitem`    | Elemento di una lista                                   | Usato per ogni elemento all'interno di una lista.                                                           | `accessibilityRole="listitem"`    | `<li>`                                |
| `checkbox`    | Checkbox selezionabile                                  | Usato per opzioni a selezione singola (come una casella di spunta).                                         | `accessibilityRole="checkbox"`    | `<input type="checkbox">`             |
| `radio`       | Opzione di una selezione esclusiva                      | Usato per gruppi di opzioni selezionabili solo una volta (radio buttons).                                   | `accessibilityRole="radio"`       | `<input type="radio">`                |
| `combobox`    | Selettore che combina un campo di testo e una lista     | Usato per campi di selezione avanzata (come un campo con una lista di opzioni).                             | `accessibilityRole="combobox"`    | `<select>`                            |
| `slider`      | Barra di scorrimento                                    | Usato per rappresentare un slider per regolare valori come volume, luminosità, ecc.                         | `accessibilityRole="slider"`      | `<input type="range">`                |
| `switch`      | Interruttore on/off                                     | Usato per toggle switch (attivo/inattivo) simile alla checkbox, ma con il comportamento di un interruttore. | `accessibilityRole="switch"`      | `<input type="checkbox">` (con stile) |
| `alert`       | Messaggio importante o avviso                           | Usato per avvisi urgenti o notifiche.                                                                       | `accessibilityRole="alert"`       | `<div role="alert">`                  |
| `alertdialog` | Dialogo di avviso                                       | Usato per modali di avviso urgente o importante.                                                            | `accessibilityRole="alertdialog"` | `<dialog>`                            |
| `dialog`      | Modale di dialogo                                       | Usato per finestre di dialogo o modali generici.                                                            | `accessibilityRole="dialog"`      | `<dialog>`                            |
| `form`        | Contenitore di un modulo                                | Usato per raggruppare i campi di un modulo.                                                                 | `accessibilityRole="form"`        | `<form>`                              |
| `status`      | Informazioni di stato                                   | Usato per messaggi che aggiornano lo stato del sistema o applicazione.                                      | `accessibilityRole="status"`      | `<div role="status">`                 |
| `toolbar`     | Barra degli strumenti                                   | Usato per una barra degli strumenti, come una toolbar o una barra di azioni.                                | `accessibilityRole="toolbar"`     | `<div role="toolbar">`                |
| `tree`        | Struttura a albero (gerarchica)                         | Usato per rappresentare strutture ad albero, come navigazione a livelli o categorie.                        | `accessibilityRole="tree"`        | `<ul role="tree">`                    |
| `tabpanel`    | Pannello contenuto di una tabella                       | Usato per il contenuto di un pannello all'interno di una tabella o di una scheda.                           | `accessibilityRole="tabpanel"`    | `<div role="tabpanel">`               |


🎯 1️⃣ Ruoli più comuni (quelli che userai davvero spesso)
✅ button

Elemento cliccabile che esegue un’azione.
👉 Usalo per TouchableOpacity, Pressable, ecc.

✅ link

Elemento che naviga verso un’altra pagina.
👉 Se fai un bottone che cambia route → meglio link.

✅ heading

Intestazione (equivalente a <h1>, <h2>, ecc.).
Si usa insieme a aria-level (su web).

✅ img

Elemento immagine.

✅ list

Contenitore di lista (tipo <ul>).

✅ listitem

Elemento dentro una lista.

✅ navigation

Sezione di navigazione (menu principale).

✅ main

Contenuto principale della pagina.

✅ form

Contenitore di form.

✅ checkbox

Checkbox selezionabile.

✅ radio

Radio button.

✅ switch

Toggle on/off.

✅ tab

Singola tab.

✅ tablist

Contenitore delle tab.

✅ tabpanel

Contenuto di una tab.
